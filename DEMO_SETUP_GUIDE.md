# 🚀 Demo Setup Guide - Quintype Performance Analyser

## ⚡ Quick Start (5 minutes)

### 1. **Start the Backend Server**
```bash
cd backend
node app-minimal.js
```
**Expected Output:** `Server running on port 3000`

### 2. **Open the Frontend**
```bash
open http://localhost:3000/stable-seo-analyzer.html
```

### 3. **Test URLs for Demo**
- **Fast Site:** `https://google.com` (High performance)
- **Medium Site:** `https://example.com` (Some issues)
- **Slow Site:** `https://httpbin.org/html` (Multiple issues)
